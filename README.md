# Calculater

A simple Javascript Based, beginner-friendly calculator built with vanilla JavaScript, HTML, and CSS.

This small project is designed to help beginners learn basic DOM manipulation and event handling in JavaScript by building a functional calculator.

Features
- Basic arithmetic: addition, subtraction, multiplication, and division
- Clear and delete (backspace) functions
- Responsive and minimal UI for easy experimentation
- Easy to read, commented code for learning

Demo
- Open `index.html` in your browser to try the calculator.
- Or serve the project locally with a simple server:
  - Python 3: `python -m http.server 8000`
  - Node (http-server): `npx http-server`

Usage
1. Click on the number and operator buttons to build an expression.
2. Press `=` to evaluate the result.
3. Use `C` to clear the display and `←` (or DEL) to remove the last digit.

How to run locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Sahaab-Ji/calculater.git
   cd calculater
   ```
2. Open `index.html` in your browser or run a simple static server as shown above.
